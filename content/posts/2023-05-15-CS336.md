## Byte-Pair Encoding Tokenizer

### Unicode
- size: >150k
- chr(0) return '\x00' 是一个不可见控制字符，Unicode中索引为0对应的字符，不可见但是有字节值
- 标准Unicode词表太大而且由于是一个字符对应一个值，所以概率小的字符和概率大的字符都占一个值，导致词表是稀疏的，没有进行压缩

### BPE
思路是说：词表规定了整数值到bytes的映射，输入文本得到整数值列表。

文本中会有special_token（例如规定文本的结束），所以初始词表大小为`256+len(special_token)`，接下来要对相邻的byte进行合并得到新的token_bytes，并加入词表。

还有一点是单词之间应当是自然的边界，所以现基于单词进行**预分词**应当既保留了语义也不会损失太多压缩性能，不过这种重新组织信息的过程可能会影响后续的建模和表达。

预分词之后统计频次，进行**合并**更新vocab和merge_list，需要注意合并之后序列发生了变化，重新统计相邻字节对频次，但不需要重新完整计算，只需考虑发生合并的哪些序列。

```python
    # pretoken_counts记录seq的频率，pair_counts记录pair的频率，pair_to_seqs记录每个pair对应哪些seq，每次merge之后这三部分都要更新
    # 1. 先找出频率最高的pair，根据pair_to_seqs得到受到影响的seq和这些seq对应的freq
    # 2. 由于这些seq是要被合并的，所以要在pretoken_counts里删除，合并之后的seq要进来
    # 3. pair_counts里要把这些old_seq包含的pair对应的freq都减掉，然后加入合并之后的new_seq包含的pair的频率
    # 4. pair_to_seqs里要把old_seq包含的pair对应的seq去掉，加入new_seq包含的pair对应的seq
    pretoken_counts = build_pretoken_counts(text, special_tokens)
    pair_counts, pair_to_seqs = count_adjacent_pairs(pretoken_counts)
    merges = []

    for i in range(num_merges):
        if not pair_counts:
            break

        # 这一步保留最原始、最容易理解的写法：
        # 只优化 pair 计数的维护，不优化“找最大 pair”。
        target_pair = max(pair_counts.items(), key=lambda x: (x[1], x[0]))[0]

        affected_seqs = list(pair_to_seqs.get(target_pair, ()))
        affected_freqs = {seq: pretoken_counts[seq] for seq in affected_seqs}
        merged_seq_counts = {}

        for seq in affected_seqs:
            del pretoken_counts[seq]

        for seq, freq in affected_freqs.items():
            old_seq_pair_counts = count_pairs_in_sequence(seq)
            for pair, count_in_seq in old_seq_pair_counts.items():
                new_count = pair_counts[pair] - count_in_seq * freq
                if new_count > 0:
                    pair_counts[pair] = new_count
                else:
                    pair_counts.pop(pair, None)

                pair_to_seqs[pair].discard(seq)
                if not pair_to_seqs[pair]:
                    pair_to_seqs.pop(pair, None)

            new_seq = merge_sequence_once(seq, target_pair)
            merged_seq_counts[new_seq] = merged_seq_counts.get(new_seq, 0) + freq

        for new_seq, freq in merged_seq_counts.items():
            pretoken_counts[new_seq] = pretoken_counts.get(new_seq, 0) + freq

            new_seq_pair_counts = count_pairs_in_sequence(new_seq)
            for pair, count_in_seq in new_seq_pair_counts.items():
                pair_counts[pair] = pair_counts.get(pair, 0) + count_in_seq * freq
                if pair not in pair_to_seqs:
                    pair_to_seqs[pair] = set()
                pair_to_seqs[pair].add(new_seq)

        vocab[256 + len(special_tokens) + i] = target_pair[0] + target_pair[1]
        merges.append(target_pair)
```
